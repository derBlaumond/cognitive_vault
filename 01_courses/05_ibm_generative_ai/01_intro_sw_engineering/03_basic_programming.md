## 2. Query and Assembly Programming Languages

#### 고급 언어 (High-Level Languages)

- 사람이 이해하기 쉬운 영어 문법과 구조를 사용한다. 예: SQL, Python.
    
- 코드 작성과 디버깅이 빠르고, 유지보수가 용이하다.

#### 저급 언어 (Low-Level Languages)

- 기계어에 가까운 형태로, **심볼(symbol)** 로 명령을 표현한다. 예: Assembly 언어(ARM, MIPS, x86 등).
    
- 하드웨어 구조에 밀접히 연결되어 CPU마다 다른 버전 존재.
#### Query Language (질의 언어)

- 데이터베이스(DB)에 **질의(Query)** 를 보내 데이터를 검색하거나 조작하는 언어.
    
- **CRUD(Create, Read, Update, Delete)** 연산 수행 가능.

### Assembly Language (어셈블리 언어)

- 저급 언어로, 0과 1로 된 기계어를 사람이 이해하기 쉽게 표현한 언어. **어셈블러(assembler)** 가 기계어로 변환한다.

- 고급 언어보다 단순하지만, 하드웨어와 밀접하며 CPU 아키텍처마다 다르다.
