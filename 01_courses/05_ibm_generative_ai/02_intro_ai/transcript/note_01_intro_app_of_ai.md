# Introduction to Artificial Intelligence (AI)

## 1. Introducing AI

### 개념 정리

AI(Artificial Intelligence, 인공지능)는 인간의 지능적 사고 과정을 컴퓨터 시스템이 **시뮬레이션(simulation)** 하는 기술이다.  
즉, 학습(Learning), 추론(Reasoning), 문제 해결(Problem Solving), 의사결정(Decision Making) 등 인간이 수행하던 사고 기능을 **알고리즘과 데이터로 재현**하는 것을 말한다.

AI는 **단순한 자동화(Automation)** 수준부터, **복잡한 딥러닝(Deep Learning)** 및 **신경망(Neural Networks)** 모델까지 다양한 형태로 발전했다.

### 역사적 흐름

- **1950년대:** 앨런 튜링(Alan Turing)이 ‘튜링 테스트(Turing Test)’ 제안 → 기계 지능의 개념 시작
    
- **1956년:** 존 매카시(John McCarthy)가 “Artificial Intelligence” 용어를 최초로 사용
    
- **1960–70년대:** ELIZA, SHRDLU 같은 초기 대화형 프로그램과 전문가 시스템(Expert Systems) 등장
    
- **1980–1990년대:** 머신러닝(Machine Learning) 개념 확립, 신경망(Neural Network) 도입
    
- **2000년대:** 딥러닝(Deep Learning) 기술 발전
    
- **2010년 이후:** NLP(자연어 처리)·컴퓨터 비전·자율주행 등 산업 전반으로 확산
    

### 핵심 기술 기반

- **인터넷(Internet):** 정보 접근성 확대
    
- **분산 컴퓨팅(Distributed Computing):** 대규모 데이터 처리 가능
    
- **IoT:** 연결된 장치에서 방대한 데이터 생성
    
- **소셜 미디어:** 비정형 데이터(Unstructured Data) 폭증
    

이 조합이 오늘날 AI 혁신의 기반이 되었다.

### 인간 지능과의 관계

- **인간의 타고난 지능(Innate Intelligence)** 은 생명체가 자율적으로 발달하고 진화하는 자연적 능력.
    
- 반면 **AI의 학습능력은 인간이 설계한 것**으로, 데이터를 입력받아 스스로 학습하도록 만든다.
    

AI 학습 방식은 다음 세 가지로 나뉜다:

1. **지도학습(Supervised Learning)** – 입력과 정답이 모두 주어진 데이터를 학습
    
2. **비지도학습(Unsupervised Learning)** – 정답 없이 패턴을 스스로 발견
    
3. **강화학습(Reinforcement Learning)** – 보상과 벌점을 통해 최적 행동을 학습
    

### AI의 분류 (Strength 기준)

|유형|설명|예시|
|---|---|---|
|**약한 AI (Weak / Narrow AI)**|특정 영역에 특화된 AI|번역기, 스팸 필터, 추천 엔진|
|**강한 AI (Strong / General AI)**|여러 영역에서 인간 수준으로 학습·응용 가능|범용 인공지능(AGI) – 현재 연구 단계|
|**초지능 (Super / Conscious AI)**|인간을 능가하는 자각적 AI|자율로 사고·감정 판단 (이론적 개념)|

### AI의 학제적 특성

AI는 여러 학문의 융합이다:

- **컴퓨터공학·전기전자공학:** 하드웨어·소프트웨어 구현
    
- **수학·통계학:** 모델링 및 성능 측정
    
- **심리학·언어학:** 인간 사고·언어 이해 구조 참고
    
- **철학:** 지능·의식의 본질, 윤리적 방향 제시

## 2. Artificial Intelligence vs. Augmented Intelligence

### 개념 비교

- **Artificial Intelligence (AI):**  
    인간의 역할을 **대체(replace)** 하는 지능.  
    예: 자율주행 시스템, 자동 번역기
    
- **Augmented Intelligence (AI):**  
    인간과 기계가 **협력(cooperate)** 하여 능력을 **보강(augment)** 하는 형태.  
    예: 운전 보조 시스템, 음성 기반 내비게이션, 화면 낭독기
    

### 예시 — 운전 사례

- 운전자는 **인간 지능(human intelligence)** 으로 기본 조작 수행
    
- 고속도로에서 **자율주행 기능(artificial intelligence)** 이 차선을 유지하고 속도를 제어
    
- 시내 주행에서는 **보조 인텔리전스(augmented intelligence)** 로 충돌 방지·사각경 감지 지원
    

### 인간 vs. 기계의 강점 비교

|구분|인간의 강점|기계의 강점|
|---|---|---|
|학습|일반화 능력, 창의성, 감정 이해|방대한 데이터 학습 속도|
|작업|문제 해결, 사회적 상호작용|반복 작업, 피로 없음|
|의사결정|직관·상황 판단|정확도·일관성|

→ **증강지능(Augmented Intelligence)** 은 양쪽의 강점을 결합한 최적의 형태다.  
이는 인간의 창의성과 기계의 계산 능력이 협력하는 “공진화적 지능”으로 간주된다.

## 3. Introducing Generative AI and Its Use Cases

### 개념

**Generative AI (생성형 인공지능)** 은 기존 데이터를 분석하는 전통적 AI와 달리,  
**새로운 콘텐츠를 창조(create)** 하는 기술이다.

→ 텍스트, 이미지, 음악, 영상 등 **새로운 형태의 데이터**를 생성 가능.

### 주요 기술

- **대형 언어 모델(LLM, Large Language Model)**
    
    - 대규모 데이터 학습으로 인간과 유사한 문장 생성
        
    - 텍스트 생성, 요약, 번역, 대화 응답 등 수행
        
- **딥러닝 기반 생성모델 (GANs, VAEs 등)**
    
    - 이미지·음악·영상 창작에 활용
        

### 활용 예시

|분야|적용 사례|
|---|---|
|**마케팅**|개인 맞춤형 광고·이메일 콘텐츠 자동 생성|
|**예술/콘텐츠 제작**|디지털 아트, 광고 영상, 음악 작곡|
|**제품 개발**|트렌드 분석 → 제품 디자인 제안|
|**의료**|수술 시뮬레이션, 맞춤형 치료 설계|
|**게임**|자동 레벨 생성, 캐릭터·세계 구축|
|**패션**|가상 피팅, 개인 스타일 추천|
|**교육**|학습자 맞춤형 교재·학습 환경 생성|

### 경제적 영향

- Goldman Sachs 보고:  
    Generative AI 도입 시 **글로벌 GDP 7% (약 7조 달러)** 성장 가능  
    생산성 향상률 **10년간 연 1.5% 증가** 예상

## 4. Different Types of AI

AI는 **기능(Functionality)** 및 **능력(Capability)** 에 따라 분류된다.

|분류|설명|주요 기능|
|---|---|---|
|**Descriptive / Diagnostic AI**|과거 데이터 분석 → 원인 파악|패턴 인식, 비교분석, 근본 원인 분석|
|**Predictive AI**|미래 예측|예측모델, 분류·군집화, 의사결정트리|
|**Prescriptive AI**|최적 행동 제시|개인화, 최적화, 사기 방지, Next Best Action|
|**Generative / Cognitive AI**|창의적 콘텐츠 생성|글쓰기, 이미지 생성, 조언 및 자동화|
|**Reactive AI**|단순 반응형|규칙기반, 실시간 응답|
|**Limited Memory AI**|과거 데이터 학습 반영|자율주행, 추천시스템|
|**Theory of Mind AI**|인간 감정·의도 인식 (연구단계)|감정 인식, 사회적 상호작용|
|**Self-aware AI**|자의식 보유 (이론적 개념)|자기진단, 자율학습|
|**Narrow AI (Weak AI)**|특정 작업 전용|스팸필터, 음성인식, 챗봇|
|**General AI (Strong AI)**|인간수준 범용 지능|다양한 도메인에 적용 가능한 AGI|

## 5. The Evolution of AI: Traditional vs. Generative

### 기존 AI (전통적 AI)

- 내부 데이터 저장소(Repository) + 분석 플랫폼 + 애플리케이션
    
- 예: 텔레콤 고객 이탈 예측 모델
    
- 문제: 자동화는 가능하지만 “새로운 지식 창출”은 불가능
    

→ “예측(Predictive)” 수준에 머무름

### 생성형 AI의 차별점

- 학습 데이터의 범위: **전 지구적 규모(Global-scale data)**
    
- LLM 기반으로 방대한 맥락 이해
    
- **프롬프팅(Prompting)** + **튜닝(Tuning)** 으로 조직 맞춤화
    
- 피드백 루프를 통해 모델 개선 지속
    

즉, **데이터 규모와 학습 구조가 완전히 다르며**,  
단순한 분석이 아닌 “생성적 추론”이 가능하다는 점이 본질적 차이이다.

## 6. Artificial Intelligence – Are We There Yet?

### 주제

“AI는 인간 수준의 지능에 도달했는가?”

### 토론 요약

- **현재 AI:** 특정 작업(체스, 계산 등)에서는 인간을 능가하지만  
    “일반지능(AGI)”은 아직 도달하지 못함.
    
- **튜링 테스트(Turing Test):**  
    인간과의 대화에서 구별 불가능하면 ‘지능 달성’이라 간주.  
    → 2014년 13세 소년 시뮬레이션 봇이 부분 통과.
    
- **현상:** 각 영역별로 ‘특화된 천재(Siloed Savants)’ 수준.  
    즉, 수학·기억·전략 등 개별 능력은 탁월하나 통합적 사고는 부족.
    
- **결론:** 아직 AGI에는 이르지 않았으나, 발전 속도는 가속 중.

## 8. AI Chatbots and Smart Assistants

### 작동 원리

1. **입력 처리:** 텍스트/음성 → Intent 및 키워드 분석
    
2. **자연어 처리(NLP):** 문맥 이해 및 응답 결정
    
3. **대화 관리(Dialogue Management):** 지식베이스에서 적절한 답 선택
    
4. **응답 생성:** 텍스트 또는 음성 형태로 결과 반환
    

### 주요 기술

- 머신러닝 + NLP + 딥러닝
    
- 음성인식(Speech-to-Text) + 텍스트생성(Text-to-Speech)
    
- 감정분석(Sentiment Analysis) 및 다국어 지원
    

### 응용 분야

|분야|예시|
|---|---|
|고객지원|자동 상담 챗봇, FAQ 처리|
|전자상거래|개인 맞춤 추천, 결제 지원|
|헬스케어|예약, 복약 알림, 증상 평가|
|교육|언어 학습, 튜터링|

## 9. What Is a Chatbot?

### 예시 — Flower Shop 시나리오

- 고객 문의: “몇 시에 여세요?” → 챗봇 자동 응답
    
- 음성/문자/웹 채널을 통해 동작
    
- 클라우드 기반 NLP 엔진이 요청을 분석 후 데이터베이스와 연동
    
- 재고 확인 → 결제 → 주문 완료까지 자동화
    

### 장점

- **시간 절약:** 반복 문의 처리 자동화
    
- **구현 용이:** 코드 없이 구축 가능
    
- **빠른 배포:** 클라우드 기반 → 1시간 내 출시 가능
